Chronus

A software to reduce U-Pb data obtained by LA-MC-ICPMS

View project on GitHub

Chronus is developed by Felipe Valença, as part of his master's project. The program was designed to handle all U-Pb data produced using a LA-MC-ICPMS system installed at the Geochronology Laboratory of the University of Brasilia (see Bühn et al, 2009). Visual Basic for Applications is the language write Chronus code. Soon the program will be available at this page and anyone will be able to collaborate with the project.
